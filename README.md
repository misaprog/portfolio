# 📊 CRP推移と臨床悪化リスクの分析

**― 医療データ分析ポートフォリオ ―**

## 1. プロジェクト概要

本プロジェクトは、入院患者の **CRP（C反応性タンパク）値の時系列変化** と
**臨床状態の悪化**の関係を分析し、
「CRPの推移から患者のリスクをどこまで推定できるか」を検証したものです。

CRPは炎症の指標として広く用いられていますが、
本分析では **単一の値ではなく「変化のしかた（トレンド）」**に注目しました。

---

## 2. 使用したデータ

* 入院患者の検査データ（CRPを含む時系列データ）
* バイタルサインおよび臨床悪化フラグ

※ 実データは個人情報保護のため非公開とし、
本リポジトリでは結果の可視化のみを掲載しています。

---

## 3. 分析の目的

臨床現場では、
「この患者は悪化しそうか？」を早期に判断することが重要です。

本プロジェクトでは以下を目的としました：

* CRPの推移パターンを分類できるか
* その分類が、実際の悪化と関連しているか
* 医療現場で使える“気づき”を得られるか

---

## 4. 分析手法の概要

1. 患者ごとにCRPの時系列データを集約
2. CRPの変化傾向に基づき、以下の3群に分類

   * **上昇群**
   * **安定群**
   * **低下群**
3. 各群における臨床悪化率を比較

分析には Python（pandas, matplotlib）および Jupyter Notebook を使用しました。

## 6. 考察

CRPの「現在値」だけでなく、
**「上昇しているかどうか」というトレンド**を見ることで、

* 悪化リスクの高い患者を早期に抽出できる可能性
* 医師・看護師の判断を補助する指標としての有用性

が示唆されました。

---

## 7. 実務への応用イメージ

この分析は、例えば以下のような用途に活かせます：

* 電子カルテやダッシュボード上で
  「CRP上昇中の患者リスト」を自動抽出
* 悪化リスクの早期アラート
* 医療AIや臨床支援システムへの特徴量として利用
